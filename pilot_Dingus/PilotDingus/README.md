# Getting Started
* Install VSCode https://code.visualstudio.com/download
* Within VSCode, install the PlatformIO IDE extension
* On menu bar, navigate `File > Open` and then select your clone folder of this repository
* Navigate to `src/main.cpp`
* On the toolbar at the bottom, click the checkmark ✔️ to test compile
* Attach Pilot Dingus to USB cable
* On the toolbar at the bottom, click the arrow → to test compile and upload to the microcontroller
